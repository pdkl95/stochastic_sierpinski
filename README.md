# stochastic_sierpinski
Interactive "Chaos Game" (IFS) fractal generator in a self-contained
HTML5 page

## Usage

Try it live on GitHub:

[https://pdkl95.github.io/stochastic_sierpinski/](https://pdkl95.github.io/stochastic_sierpinski/)
